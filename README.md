# Estimator

Simple quote/bid generator for petroleum equipment.

## Features

- Add line items with quantities and prices
- Calculate totals automatically
- Share quotes via URL (data encoded in URL)
- Export/Import JSON
- Generate PDF
- No database needed - all client-side

## Deploy to Vercel

1. Push this repo to GitHub
2. Import in Vercel
3. Deploy - done!

## Local Development

```bash
npm run dev
```

Then open http://localhost:3000
